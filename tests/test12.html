Able Player Test #12:
Video player with default closed captions

This test page was designed for testing a bug in which certain browsers display captions even if no controls attribute is present. They only do this if a <track> element has a default attribute. For additional details see Able Player Issue 5.

Update: 11/27/2015: The bug was resolved by modifying the html sourcecode of this test page rather than the javascript itself - the "default" attribute should be avoided when embedding the player in web sites. See this resource.

For additional tests see the Index of Able Player Examples.
