Able Player for YouTube

This page uses the YouTube player, but with Able Player controls.

If <track> elements are used for captions or subtitles, these take precedence over YouTube captions. Otherwise, YouTube captions and subtitles are used if available. The advantage of using local captions is they can be used to build an interactive transcript and search within videos. Currently YouTube doesn't expose specific caption data via its API so these functions are not possible using YouTube captions alone.

Also, a poster attribute can be used to replace YouTube's default poster image with an alternative image.

For additional demos see the Index of Able Player Examples.
