Audio in "Lyrics Mode"

All tracks are supported in Able Player for both video and audio, with the exception of descriptions. This example includes captions, subtitles, and chapters tracks.

The transcript (in this case, song lyrics) is provided in a captions file (in French) and a subtitles file (in English). Since the song is in French, the French subtitles are set as the default track with the default attribute. The following optional attributes affect the transcript:

  • data-lyrics-mode - forces a line break between and within captions in the transcript
  • data-transcript-title - to override default title, which is "Transcript", or "Lyrics" if the data-lyrics-mode attribute is present
