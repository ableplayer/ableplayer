Able Player Video Demo #8b:
Responsive video player with Metadata Track (type = selector)

This example has the same functionality as Demo #8 (time-synced metadata exposing content as an overlay on the video). However, in the former demo the video is positioned with fixed width and height. This is necessary to precisely position overlay content atop the video using pixel values. The current example demonstrates that this can also be accomplished in a responsive design, with overlay content positioned using percentage values.

For additional demos see the Index of Able Player Examples.
