Able Player Audio Demo #6:
Multiple audio players (one track each)

Having multiple players on a single page might be fun for producing on-the-fly mashups. However, only the first player can be operated with keystrokes, which might be confusing for users. Although Able Player supports multiple player instances, they're discouraged. Use playlists instead.

For additional demos see the Index of Able Player Examples.
