Audio description via video swap

This demo uses a separate video to deliver audio description. If Description is toggled on, the user gets the described version; otherwise they get the non-described version.

One limitation of this method, as opposed to using a WebVTT file for audio description, is that the description text is not included in the interactive transcript. This limitation can be overcome by adding a transcript of the description content in a WebVTT descriptions track.
