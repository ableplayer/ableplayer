Préférences liées au clavier

Le lecteur multimédia de cette page Web peut être utilisé à partir de n’importe quel endroit sur la page avec des raccourcis du clavier (voir la liste ci-dessous). Des rôles peuvent être assignés aux touches de modification (Shift, Alt, Ctrl) ci-dessous. Certaines combinaisons de touches pourraient entrer en conflit avec les touches utilisées par votre navigateur ou autres applications logicielles. Essayez diverses combinaisons de touches de modification pour en trouver qui fonctionnent pour vous.

Touches de modification utilisées pour des raccourcis

Raccourcis du clavier assignés actuellement

  • Alt + Ctrl + p ou barre d’espacement = Lecture/Pause
  • Alt + Ctrl + s = Redémarrer
  • Alt + Ctrl + r = Reculer
  • Alt + Ctrl + f = Avancer
  • Alt + Ctrl + v ou 1-9 = Volume
  • Alt + Ctrl + m = Activer le son/Couper le son
  • Alt + Ctrl + c = Sous-titres
  • Alt + Ctrl + d = Activer les descriptions
  • Esc = Fermer la fenêtre de dialogue ou le menu contextuel
